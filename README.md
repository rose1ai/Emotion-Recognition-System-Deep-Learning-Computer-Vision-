# Emotion-Recognition-System-Deep-Learning-Computer-Vision-
Developed an AI‑based emotion recognition system using MediaPipe’s deep‑learning Face Landmarker model to extract 468 facial landmarks from static images.

Implemented a custom rule‑based classifier to infer emotional states (Happy, Neutral, Surprised) based on geometric relationships between key facial points.

Built a complete processing pipeline including image preprocessing, face detection, landmark extraction, and emotion inference.

Visualized results by overlaying the predicted emotion directly on the processed image.

Evaluated landmark precision using Mean Landmark Error (MLE), achieving an average error of 0.004, indicating high accuracy in landmark detection.

Performed synthetic evaluation for emotion classification, achieving 80% accuracy across test samples and generating a confusion matrix for performance analysis.
